Application Overview

A high-performance storage space optimizer and directory packing utility.

Languages: English | Español | Français | Nederlands | Deutsch

Product Description

Burn to the Brim is a professional file packaging and storage optimization utility. It solves the classic bin-packing and subset-sum optimization challenge, allowing you to fit files and folders perfectly onto target storage mediums—including recordable CDs, DVDs, Blu-rays, USB flash drives, or custom-capacity virtual drives—with virtually zero wasted space.

Employing a highly optimized recursive backtracking solver, the application analyzes directory structures and file sizes to calculate near-perfect layout configurations in milliseconds. With a non-blocking multithreaded engine, the user interface remains completely fluid and interactive even when calculating solutions for very large datasets.

Key Features

  • →
    Multi-Volume Spanning: Automatically splits massive file hierarchies that exceed single-drive capacities into optimally sized volume folders.
  • →
    Cauchy Reed-Solomon PAR3 Parity: Automatically creates PAR3 parity archives for error correction, factoring overhead directly into layout checks.
  • →
    Integrity Verification & Recovery: Scan and verify volumes against parity files to identify corrupted items and perform non-destructive restoration.
  • →
    Explorer Context Menu Integration: Instantly append files and folders directly to active sessions via the Windows Explorer right-click menu.
  • →
    Hybrid GUI/CLI Engine: Seamlessly switch between a lightweight native user interface (Win32 / GTK 4) and a full command-line mode for scripting.
  • →
    Unicode & Long Path Support: Fully compatible with international character encodings and Windows UNC long paths up to 32,767 characters.
  • →
    Non-Blocking Tree Rendering: Displays large solution datasets incrementally using main-loop idle cycles to prevent GUI freezing.
  • →
    Fractional MB Capacity Precision: Supports precise capacity inputs and saving, such as custom 180 KB or 0.18 MB volumes, for accurate packing.
  • →
    Offline JSON Indexing: Generates a standardized index.json directory catalog during output, listing file attributes for offline browsing.
  • →
    Rule-Based and Semantic Grouping: Group files together on the same medium using standard wildcards, regular expressions, or context-aware semantic clustering.

Privacy & Performance Commitments

Burn to the Brim is built with user privacy, system performance, and clean software standards in mind:

  • ✗
    Strict Offline Execution (No Spyware): The application operates entirely locally on your system. It never logs, tracks, or transmits your directory paths, file names, or personal information over the network.
  • ✗
    No Adware: The application does not contain advertisements, commercial banners, or sponsored popups.
  • ✗
    Ultra-Lightweight Footprint: Statically compiled native C++ code ensures minimal memory usage and zero framework overhead.
  • ✗
    Zero System Clutter: The installer registers only what is necessary for Windows integration, and the application does not litter your registry with junk files.
  • ✗
    Clean Uninstallation: Standard uninstall removal completely cleans all program directories and shortcuts from your system.
