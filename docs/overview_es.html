Descripción del Producto

Burn to the Brim es una herramienta profesional de embalaje de archivos y optimización de almacenamiento. Resuelve el clásico desafío de optimización del problema del empaquetado (bin-packing) y la suma de subconjuntos, permitiéndole encajar archivos y carpetas perfectamente en soportes de almacenamiento—incluyendo CDs, DVDs, Blu-rays, unidades flash USB o discos virtuales con capacidad personalizada—con prácticamente cero desperdicio de espacio.

Utilizando un motor de resolución por backtracking recursivo altamente optimizado, la aplicación analiza las estructuras de directorios y los tamaños de archivos para calcular configuraciones de diseño casi perfectas en milisegundos. Con un motor multiproceso no bloqueante, la interfaz de usuario se mantiene completamente fluida e interactiva incluso al calcular soluciones para conjuntos de datos muy grandes.

Características Clave

  • →
    División Multivolumen: Divide automáticamente jerarquías de archivos masivas que superan las capacidades de un solo disco en carpetas de volúmenes de tamaño óptimo.
  • →
    Paridad PAR3 Cauchy Reed-Solomon: Crea automáticamente archivos de paridad PAR3 para la corrección de errores, considerando el coste adicional en las comprobaciones de diseño.
  • →
    Verificación de Integridad y Recuperación: Escanea y verifica volúmenes contra archivos de paridad para identificar elementos corruptos y realizar restauraciones no destructivas.
  • →
    Integración con el Menú Contextual del Explorador: Agrega instantáneamente archivos y carpetas a sesiones activas mediante el menú del botón secundario de Windows Explorer.
  • →
    Motor Híbrido GUI/CLI: Cambia sin problemas entre una interfaz de usuario nativa ligera (Win32 / GTK 4) y un modo de línea de comandos completo para scripting.
  • →
    Soporte Unicode y Rutas Largas: Totalmente compatible con codificaciones de caracteres internacionales y rutas largas UNC de Windows de hasta 32.767 caracteres.
  • →
    Renderizado de Árbol No Bloqueante: Muestra conjuntos de datos de soluciones de forma incremental utilizando los ciclos inactivos del bucle principal para evitar congelaciones de la GUI.
  • →
    Precisión de Capacidad en MB Fraccionales: Permite entradas precisas de capacidad y guardado, como volúmenes personalizados de 180 KB o 0.18 MB, para un empaquetado exacto.
  • →
    Indexación JSON sin conexión: Genera un catálogo de directorios estandarizado index.json al procesar la salida, enumerando los atributos del archivo para navegación offline.
  • →
    Agrupación Semántica y por Reglas: Agrupa archivos en el mismo medio usando comodines estándar, expresiones regulares o agrupación semántica basada en el contexto.

Compromiso de Privacidad y Rendimiento

Burn to the Brim está diseñado pensando en la privacidad del usuario, el rendimiento del sistema y los estándares de software limpio:

  • ✗
    Ejecución Estricta Fuera de Línea (Sin Spyware): El aplicativo opera completamente a nivel local. Nunca registra, rastrea ni transmite por la red sus rutas de directorios, nombres de archivos o información personal.
  • ✗
    Sin Adware: La aplicación no contiene anuncios publicitarios, banners comerciales ni ventanas emergentes patrocinadas.
  • ✗
    Consumo de Recursos Ultra Ligero: El código nativo en C++ compilado estáticamente asegura un uso mínimo de memoria y elimina cualquier sobrecarga de marcos de trabajo.
  • ✗
    Sin Basura en el Sistema: El instalador registra solo lo necesario para la integración en Windows, y la aplicación no ensucia su registro con archivos innecesarios.
  • ✗
    Desinstalación Limpia: El proceso de desinstalación estándar elimina por completo todos los directorios y accesos directos del programa de su sistema.
