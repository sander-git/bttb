产品描述

Burn to the Brim 是一款专业的文件打包与存储优化工具。它完美解决了经典的装箱问题（bin-packing）和子集和优化挑战，允许您将文件和文件夹完美地放入目标存储介质中——包括可刻录的 CD、DVD、蓝光光盘、USB 闪存驱动器或自定义容量的虚拟驱动器，几乎达到零空间浪费。

该程序采用高度优化的递归回溯求解器，可在毫秒内分析目录结构和文件大小，以计算出几近完美的布局配置。配合非阻塞多线程引擎，即使在为极大数据集计算解决方案时，用户界面也能保持完全流畅与交互性。

主要功能

  • →
    多卷分发: 自动将超过单个磁盘容量的海量文件层级分割到大小最优的卷文件夹中。
  • →
    Cauchy Reed-Solomon PAR3 冗余校验: 自动创建用于错误纠正的 PAR3 冗余归档，并将开销直接计入布局检查中。
  • →
    完整性校验与恢复: 扫描并对比校验文件以识别损坏的项目，执行无损、基于复制的安全恢复。
  • →
    资源管理器右键菜单集成: 通过 Windows 资源管理器右键菜单瞬间将文件和文件夹直接附加到活动会话中。
  • →
    混合 GUI/CLI 引擎: 在轻量级原生用户界面（Win32 / GTK 4）和用于脚本编写的完整命令行模式之间无缝切换。
  • →
    Unicode 和长路径支持: 完全兼容国际字符编码和高达 32,767 个字符的 Windows UNC 长路径。
  • →
    非阻塞树状渲染: 利用主循环空闲周期增量显示大型解决方案数据集，防止 GUI 冻结。
  • →
    MB 小数级容量精度: 支持精确的容量输入与保存，如自定义 180 KB 或 0.18 MB 卷，以实现精确打包。
  • →
    离线 JSON 索引: 在输出期间生成标准化的 index.json 目录清单，列出文件属性以便离线浏览。
  • →
    基于规则和语义的分组: 使用标准通配符、正则表达式或上下文感知的语义聚类，将相关文件归类打包在同一介质中。

隐私与性能承诺

Burn to the Brim 的设计宗旨是保障用户隐私、系统性能和干净的软件标准：

  • ✗
    严格的离线运行（无间谍软件）: 应用程序完全在您系统本地运行。它绝不会记录、跟踪或通过网络传输您的目录路径、文件名或任何个人信息。
  • ✗
    无广告软件: 本程序不包含任何广告、商业横幅或赞助弹窗。
  • ✗
    极轻量级的资源占用: 静态编译的原生 C++ 代码确保极低的内存消耗，且无任何框架开销。
  • ✗
    零系统杂乱: 安装程序仅注册 Windows 集成所必需的内容，程序不会在您的注册表中留下任何垃圾文件。
  • ✗
    干净卸载: 标准的卸载程序将从您的系统中彻底清除所有的程序目录和快捷方式。
